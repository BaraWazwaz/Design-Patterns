Detailed Description

template<typename T>
class nitron::Singleton< T >

Singleton class wrapper.

Template Parameters
TType to be wrapped as Singleton, a class has to be :
  • Privately Default Constructable.
  • Not Copy Constructable.
  • Not Move Constructable.
  • Not Copy Assignable.

  • Not Move Assignable.

    Singleton Design Pattern allows only one instance of class, to limit and control access to the wrapped class's constructor.

More found at https://refactoring.guru/design-patterns/singleton.
